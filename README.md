Dark Mode and Light Mode Toggle

This project was created with the purpose of developing a simple Dark Mode and Light Mode toggle button using JavaScript.
It demonstrates how to switch between themes dynamically, improving user experience and accessibility.

Features

Toggle between Dark Mode and Light Mode

Built with HTML, CSS, and JavaScript

Simple and responsive design

Easy to customize and integrate into other projects

How to Use

Clone or download this repository.

Open the project folder.

Run the index.html file in your browser.

Click the button to switch between dark and light themes.

Purpose

The main goal of this project is to practice and demonstrate JavaScript DOM manipulation and theme switching logic.
